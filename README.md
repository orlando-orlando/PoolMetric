# PoolMetric
# Software de modelado de Ingeniería Hidráulica para Piscinas

## Origen del Proyecto y Antecedentes

Este proyecto se originó como una **herramienta personal de ingeniería** desarrollada para asistir en el diseño técnico y análisis de sistemas acuáticos como piscinas, spas y cuerpos de agua similares.

El concepto inicial comenzó en **2020** como una iniciativa personal para mejorar y automatizar cálculos de ingeniería hidráulica y térmica que se realizan de forma repetitiva en el diseño de instalaciones acuáticas.

Las primeras versiones funcionales fueron desarrolladas como **hojas de cálculo en Excel**, creadas exclusivamente para uso personal con fines de productividad, verificación técnica y apoyo en procesos de diseño de ingeniería.

El objetivo de la herramienta original era consolidar múltiples cálculos de ingeniería dentro de un flujo de trabajo estructurado que permitiera realizar de forma sistemática:

* Cálculo de volumen de cuerpos de agua
* Cálculo de flujo hidráulico
* Análisis de tasa de rotación del sistema
* Estimación de Carga Dinámica Total (TDH)
* Apoyo para selección de motobombas
* Cálculo de pérdidas de calor del agua
* Dimensionamiento de equipos de calentamiento

Las primeras versiones fueron desarrolladas **de forma independiente**, utilizando únicamente principios de ingeniería disponibles públicamente, incluyendo fundamentos de hidráulica, mecánica de fluidos y transferencia de calor.

Todos los cálculos implementados en este proyecto se basan en **leyes físicas y fórmulas de ingeniería ampliamente conocidas y documentadas**, tales como:

* Mecánica de fluidos
* Ecuaciones de pérdidas de carga en tuberías
* Principios de transferencia de calor
* Ecuaciones de fricción en tuberías
* Pérdidas menores por accesorios hidráulicos

No se utilizaron fórmulas propietarias, metodologías confidenciales ni información técnica protegida perteneciente a ningún empleador o a terceras organizaciones durante el desarrollo de este proyecto.

---

## Evolución hacia Plataforma de Software

Entre **2023 y 2025**, el proyecto evolucionó desde herramientas basadas en hojas de cálculo hacia una **plataforma de software desarrollada con tecnologías web modernas**.

La transición a software tuvo como objetivo:

* Mejorar la experiencia de usuario
* Automatizar flujos de cálculo complejos
* Permitir trazabilidad completa de los cálculos de ingeniería
* Superar las limitaciones técnicas de las hojas de cálculo
* Incorporar modelos hidráulicos y térmicos más avanzados

La implementación actual del software es una **reescritura completa del sistema**, desarrollada desde cero.

El código fuente del proyecto **no contiene código proveniente de hojas de cálculo anteriores ni de sistemas desarrollados por terceros**.

## Alcance de Ingeniería de la Herramienta

El software actualmente incluye módulos de cálculo para:

### Diseño Hidráulico

* Cálculo de volumen de piscinas y cuerpos de agua
* Configuración de tasa de rotación
* Cálculo de flujo del sistema
* Selección de diámetro de tuberías
* Cálculo de velocidad de flujo
* Pérdidas menores por accesorios hidráulicos (codos, tees, válvulas, reducciones)
* Modelado de Carga Dinámica Total (TDH)
* Apoyo en selección de motobombas

### Componentes del Sistema Acuático

* Sistemas de filtración
* Equipos de sanitización
* Accesorios hidráulicos empotrables
* Modelado hidráulico de infinity edge
* Modelado de múltiples cuerpos de agua conectados

### Análisis Térmico

* Cálculo de pérdidas de calor por:

  * Evaporación
  * Convección
  * Radiación
  * Conducción
  * Transmisión
  * Tubería
  * Infinity
  * Canal perimetral
* Estimación de demanda térmica mensual
* Dimensionamiento de equipos de calentamiento

### Herramientas de Apoyo al Diseño

* Generación de memorias de cálculo
* Explosión de materiales
* Modelado del sistema hidráulico
* Herramientas geométricas para estimar áreas a partir de planos o imágenes escaladas

## Filosofía de Ingeniería del Proyecto

El objetivo de esta herramienta es proporcionar a ingenieros y diseñadores un **entorno de cálculo transparente basado en principios físicos**, donde las suposiciones de diseño, fórmulas utilizadas y resultados intermedios puedan ser revisados y verificados.

El software está diseñado como una **herramienta de apoyo al cálculo**, no como sustituto del criterio profesional del ingeniero.
